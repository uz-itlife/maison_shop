📋 Как использовать базу товаров на сайте
После добавления товаров — нажмите ⬇ Экспорт JSON и сохраните файл как data/products.json рядом с index.html. Сайт автоматически загрузит товары из этого файла.
Структура папок:
project/ ├── index.html ├── admin.html ← эта страница ├── css/style.css ├── js/script.js ├── data/ │ └── products.json ← база товаров (экспортируй сюда) └── images/ └── products/ ├── dress-1.jpg └── blazer-1.jpg
